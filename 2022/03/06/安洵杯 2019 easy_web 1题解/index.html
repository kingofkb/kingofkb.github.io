安洵杯 2019 easy_web 1题解

打开题目发现url

http://4c1b283f-95d8-4289-8768-e8a796e68354.node4.buuoj.cn:81/index.php?img=TXpVek5UTTFNbVUzTURabE5qYz0&cmd=

猜测TXpVek5UTTFNbVUzTURabE5qYz0是base64加密，这里有一个小坑，相信大家都用过这个网站CTF在线工具，但是这个网站是解不了这个base64码的，不知道具体原因是什么，我这里编写脚本进行解码：

<?php
$t="TXpVek5UTTFNbVUzTURabE5qYz0";
echo(base64_decode($t));
echo("\n");
$t="MzUzNTM1MmU3MDZlNjc=";
echo(base64_decode($t));
?>

输出为

if ((string)$_POST['a'] !== (string)$_POST['b'] && md5($_POST['a']) === md5($_POST['b']))

绕过MD5判断，注意HackBar会自动解码一次，不能正常绕过，复制payload在burpsuite里传值。

bin   dev  flag  lib    media  opt   root  sbin  sys  usr

找到flag，cat被ban，常规用cut绕过，但-被也ban，这里可以用base64 /flag绕过，得到ZmxhZ3s1MzExMTNlMS03MmQxLTQ3M2ItYjBhZi04YzE3NDhiMzgxOGR9Cg==解码得到flag:flag{531113e1-72d1-473b-b0af-8c1748b3818d}。
也可以用ca/t /flag绕过cat的正则判断，个人理解：/t在正则会被识别为tab，但执行命令时又当做t，达到绕过的效果。
